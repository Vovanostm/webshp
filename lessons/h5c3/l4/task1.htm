Для того, чтобы задать стиль с помощью такого подключения, надо прямо в тег добавить атрибут style

Link

И, наконец, гвоздь нашей рограммы - подключение стилей через линковку. То есть, вы пишите стили в отдельном файле, а в html подключаете с помощью строки <link rel="stylesheet" href="style.css">.

Здесь атрибут rel определяет отношения между текущим документом и документом, на который ведет ссылка. А атрибут href указывать путь до файла со стилями!
