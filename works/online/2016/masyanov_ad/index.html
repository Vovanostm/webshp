Что такое Windows Phone

Microsoft в очередной раз пытается занять долю на мобильном рынке, ранее они безуспешно пытались сделать это с КПК на Windows Mobile, а сейчас продвигают свою ОС Windows Phone. В них используется дизайнерская система Modern UI. Microsoft определённо продолжает дело Казимира Малевича, ведь всё что мы можем изменить – это цветовую гамму квадратиков.

Плюсы Windows Phone

Из-за примитивности интерфейса, эта OS не тормозит даже на бюджетных устройствах, и разобраться в ней сможет даже человек, далёкий от мира IT-технологий, чего не скажешь об android. Конечно же, в Windows Phone Store пока что нет такого количества приложений, как в Google Play или в App Store.

Windows 10 — новое начало

Windows 10 стала единственной программной платформой Microsoft, управляющей различными устройствами: от микроконтроллеров и до больших серверных систем. Только вдумайтесь: одно ядро для ПК, консолей, смартфонов и т. д.

Создание приложений под Windows Phone

Портирование Android приложений

Сегодня существует удобная утилита на основе Project Astoria, позваляющая портировать андроид-приложения на Windows Phone, не имеющие привязки к Google сервисам.

Языки программирования

Для создания предложений под Windows Phone используется C-подобный C# и XAML, который является основой для дизайна пользовательского интерфейса для Windows Phone.

Фрэймворки

На выбор разработчикам под Windows Phone даётся два фрэймворка: либо Silverlight, либо XNA.
